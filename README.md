Automatización de contenido para YouTube Shorts mediante Python y API Data v3. Proyecto desarrollado como desafío personal para optimizar flujos de trabajo y aprender integración de sistemas.

# YouTube Automation: Data Mundial
Este proyecto nació como un **desafío personal** para explorar la automatización de procesos creativos y la gestión de APIs de Google.

### 🚀 ¿Qué hace este script?
- **Automatización de flujo:** Genera y prepara contenido (Shorts) de forma programada.
- **Integración de API:** Utiliza YouTube Data API v3 para la gestión de metadatos y carga.
- **Optimización:** Reduce tareas repetitivas mediante lógica de Python.

### 🧠 Aprendizaje Activo
Lo desarrollé con el objetivo de aprender a manejar credenciales OAuth2 y entender la escalabilidad de procesos digitales, demostrando mi capacidad para aprender herramientas complejas de forma autodidacta.
